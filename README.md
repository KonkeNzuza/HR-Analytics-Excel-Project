HR Analytics — Excel

Business Problem
Which departments have the highest employee turnover and 
what is the salary profile of employees who left?

Tools Used
- Microsoft Excel — data cleaning, analysis and dashboard

Key Findings
- Sales department has the highest turnover rate at 83.87%
- Sales also has the lowest engagement score at 3.82
- IT/IS and Software Engineering lose their highest earners 
  to voluntary exits — suggesting competitor poaching
- Production has the most exits in absolute numbers 
  but salary is not the driver — culture is more likely the cause
- Overall company turnover rate is 33.44%

Excel Skills Demonstrated
- Data cleaning — TRIM, LEFT, MID, FIND, DATEDIF, IFERROR
- Calculated columns — Tenure Years, Salary Band, 
  Performance Category, Attrition Type
- Lookup functions — XLOOKUP, VLOOKUP, INDEX MATCH
- Aggregation — COUNTIF, COUNTIFS, SUMIF, AVERAGEIF
- Pivot Tables and Pivot Charts
- Interactive slicer connected to multiple pivot tables
- KPI dashboard with dynamic formulas
